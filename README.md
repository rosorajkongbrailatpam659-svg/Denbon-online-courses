# Denbon Online Courses

## About Project
Denbon Online Courses is a simple online course website created using HTML, CSS, and JavaScript. It provides information about different courses and helps users explore available learning content.

## Features
- Home page
- About page
- Courses page
- Contact form
- Privacy and Terms pages
- Responsive design
- Simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- google Sheets

## How to Run
1. Download the project.
2. Open the folder.
3. Open `index.html` in a browser.

## Project Structure
- HTML files for web pages
- CSS folder for design
- Image folder for images
- JavaScript file for functionality
